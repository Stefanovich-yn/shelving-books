# shelving-books
Для хранения данных о книгах используется одномерный массив целых чисел (int[]), позволяющий последовательно хранить значения толщины каждой книги.

В коде определены константы DEFAULT_SHELF_WIDTH (10) и MIN_THICKNESS (1). Это позволяет легко изменять параметры системы (например, максимальную вместимость полки) в одном месте.

Решение структурировано с использованием статических методов:
* inputBooks — обеспечивает ввод данных пользователем и их строгую валидацию (проверка на положительное количество книг, контроль минимальной толщины и сопоставление габаритов книги с шириной полки);
* arrangeBooks — реализует логику распределения книг по полкам: книги добавляются на текущую полку до тех пор, пока их суммарная толщина не превысит лимит, после чего открывается новая полка. Метод также визуализирует состав каждой полки и рассчитывает итоговую статистику.

Интерфейс реализован в виде текстового меню на базе цикла while(true) и конструкции switch-case, обеспечивающие интерактивность и непрерывную работу программы до выбора пользователем пункта «Выход».
